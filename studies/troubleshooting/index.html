1 Brief

This week we will gain hands-on experience with using the browser development console introduced in the lecture. Specifically, we will investigate the internals of the Rolex Sydney Hobart Yacht Race:

https://www.rolexsydneyhobart.com

The rest of the tutorial sheet outlines some activities for investigating the website. Use the final pages of the document to record any of your findings while completing the activities. Once you have a reasonable understanding of the webpage, scribble a container diagram you imagine reflects the implementation.

2 Activity: Exploring Endpoints

Most webpages use API requests for data to populate the contents of a webpage. Explore the website and its features. Use the developer console to identify the various API end-points, include information such as the URL, HTTP method, request headers, request data, etc. You can write these end-points down or add them to Postman. Also note any pages you find which do not use API requests to populate the content.

3 Activity: Explore Tooling

Explore the folder structure of the website. Try and use this folder structure to identify the key frameworks and libraries that are used. If you are able, use other developer console features to try and identify what backend frameworks might be utilised.

4 Activity: Persistent Data

In the Yacht Tracker, you can favourite yachts and only show them on the map. Once you favourite a yacht, it stays favourited when you refresh or close the browser and open it again.

  • Where is this favourite data stored?

  • How do you remove the favourites using the browser developer tools?

5 Activity: AWS Services

The Rolex Sydney to Hobart website reaches a load of a peak of 15,000 international viewers concurrently during the beginning and end of the race. They are mostly refreshing the tracking screen. What kind of AWS infrastructure would you recommend to handle that load that is also budget friendly?

6 Bonus Activity: Bug Fixing

Changes are not showing up in production that you just deployed, and you made changes to some styles in rshyr-rolex.css — what would you need to do to get the changes to show up?
